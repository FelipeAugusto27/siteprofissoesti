Desenvolvedor Web

Como o próprio nome diz, o desenvolvedor web desenvolve e cria websites (como este, portando se eu recebesse por fazer esse site, eu seria um web dev?). Num mundo em que vemos cada vez mais digitalização, o uso de websites, e o número dos mesmos, cresce exponencialmente.

Para trabalhar nesta área, é necessário possuir conhecimento das linguagens de marcação HTML e CSS (que eu também estou usando agora), mas também de linguagens de programação como Javascript para trabalhos mais complexos.
